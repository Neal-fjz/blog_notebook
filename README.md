# 项目说明

> 本项目使用微服务进行开发(为方便后期添加不同功能模块), jdk 版本1.8, 需要使用到的开源技术如下:
1. nacos :服务注册与发现中心
2. spring cloud gateway:网关架构
3. spring cloud:微服务架构
4. sa-token:认证模块
5. mysql:数据库
6. redis:缓存中间件
7. open-feign: 远程调用客户端
8. mybatis-plus: 对数据库的增删改查操作

* 要求统一使用最外层 pom 统一包版本号, 如果需要添加新的模块, 只需要在 pom 中添加依赖即可
>目前先分为两个模块:1.用户注册登录与权限管理 2.博客管理

需要实现功能:用户注册\登录\权限管理\密码修改
